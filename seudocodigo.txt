idAhora
codigo de python de una funcion llamada idAhora()
que retorne en base a la fecha (en formato dd/mm/yyyy) 
y hora actual (en formato hh:mm:ss)  
retorne un string sin"/" y sin ":"
osea si la fecha es 23/12/2023 y la hora es 09:01:23 
entonces el retorno seria "23122023090123" 
tener encuenta que fecha y hora el formato en decimal 
osea si la hora es 9:1:23 etonces es 090123 
lo mismo para la fecha
